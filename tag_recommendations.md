# noteタグ戦略 & 記事別おすすめタグ

## 現状の問題点

1. **#自分** を多くの記事に使っているが、検索ボリュームに対して競合が多すぎ、内容との関連も薄い
2. **#占い師** タグが1記事にしか付いていない → 仕事に繋がるシグナルが弱い
3. タグ数が少ない記事がある（4個だけの記事も）→ 発見機会の損失
4. **#スピ** と **#スピリチュアル** の重複、**#運** のような曖昧タグがある
5. 高ボリュームの汎用タグ（#エッセイ、#人間関係 など）がほぼ使われていない
6. 「鑑定してほしい」と思わせるタグ導線がない

---

## タグ戦略：3層構造

### 第1層：固定タグ（全記事に必ずつける）2〜3個
| タグ | 記事数 | 目的 |
|------|--------|------|
| #占い | 約17.5万件 | コアカテゴリ |
| #四柱推命 | ニッチ | 専門性のシグナル |
| #占い師 | 中規模 | **仕事に繋げる最重要タグ**（プロであることを示す） |

### 第2層：カテゴリタグ（記事テーマに応じて選ぶ）2〜3個
| タグ | 記事数 | 使うべき記事テーマ |
|------|--------|-------------------|
| #人間関係 | 約19.5万件 | 相性・テイカー・恋愛系 |
| #恋愛 | 高ボリューム | 恋愛・出会い・モテ系 |
| #エッセイ | 約165万件 | 体験談・考察系（ほぼ全記事に使える） |
| #自己分析 | 中ボリューム | MBTI・性格・自分を知る系 |
| #生き方 | 高ボリューム | 人生・運気・死生観系 |
| #スピリチュアル | 約18.7万件 | スピ批判・占いの立場表明系 |
| #悩み相談 | 中ボリューム | 占いのタイミング・相談系 |

### 第3層：記事固有タグ（その記事だけの固有トピック）1〜3個
- 記事の具体的なキーワード（#MBTI、#大殺界、#丙午 など）
- トレンドワード（#2026年、#Netflix など）

## まとめ：仕事に繋げるための最重要ポイント

1. **#占い師 を全記事に必ずつける** → 「この人はプロなんだ」と認識させる
2. **#人間関係（19.5万件）、#エッセイ（165万件）** → 占いに興味がなくても流入してくる大ボリュームタグ
3. **#MBTI、#大殺界、#丙午** → 2026年のトレンドワードを押さえる
4. **#悩み相談、#人生相談** → 「相談したい」と思っている潜在顧客にリーチ
5. **タグは7〜9個が目安** → 少なすぎると発見機会を逃す、多すぎると焦点がぼやける
