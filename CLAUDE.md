# 執筆ルール

## 読みやすさの原則

- **ターゲット層**: 中学生〜高校生でも理解できるレベルを目指す
- 難しい内容も噛み砕いて、身近な例えを使って説明する
- 一文は短めに。長くなりそうなら分ける

## 文体

- **基本は「だ・である」口調**
- 敬語（です・ます）はCTA（行動喚起）の部分でのみ使用可
  - 例: 「▼気になる方は、お気軽にご相談ください」
- 口語的な表現（「〜って」「〜じゃない？」など）は適度に使ってOK

## 専門用語の扱い

- **専門用語を出す時は必ず説明を入れる**
- 読み仮名が必要なものはカッコ書きで補足
  - 例: 四柱推命（しちゅうすいめい）
  - 例: 空亡（くうぼう）
- 初出時に簡潔な説明を添える
  - 例: 「MBTIとは、性格を16タイプに分ける性格診断のこと」
  - 例: 「属性とは、木・火・土・金・水の5種類」

# 作業ルール

## フィードバック反映のフロー

1. 記事の構成や内容についてフィードバックをもらったら、まず記事を修正する
2. そのフィードバックを「抽象化した学び」として言語化する
3. 抽象化した学びをこのCLAUDE.mdの執筆ルールに反映する

これにより、同じ指摘を繰り返さずに済み、執筆の質が継続的に向上する。
