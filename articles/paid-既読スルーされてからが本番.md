## 同じメッセージでも、"嬉しい"と"重い"が真逆になる

これ、鑑定をしていて本当に実感していること。

たとえば「今日どうだった？」というLINE。

ある人にとっては、「気にかけてくれてる、嬉しい」。
別の人にとっては、「毎日報告しなきゃいけないの？重い」。

**同じ一文なのに、受け取り方が真逆。**

なんでこうなるかというと、人によって**心地いいコミュニケーションの距離感がまるで違う**から。

四柱推命で人を5タイプ（木・火・土・金・水）に分けると、このLINEの受け取り方の違いがものすごくはっきり見える。

## 5タイプの概要

四柱推命では、生まれた日の「日干」で人を大きく5つの属性に分ける。

- **木タイプ（甲・乙）:** 自立心が強い。自分のペースが大事。
- **火タイプ（丙・丁）:** 感情豊か。ノリと勢いを大事にする。
- **土タイプ（戊・己）:** 安定志向。誠実さと安心感を大事にする。
- **金タイプ（庚・辛）:** こだわりが強い。筋が通っていることを大事にする。
- **水タイプ（壬・癸）:** 自由。束縛されないことを大事にする。

**相手のタイプの調べ方:**
1. 「四柱推命 命式 無料」で検索
2. 相手の生年月日を入力
3. 「日干」の欄を見る → 甲or乙＝木、丙or丁＝火、戊or己＝土、庚or辛＝金、壬or癸＝水

これだけ。3分で分かる。

で、タイプが分かったら、**そのタイプ専用のLINEの送り方**がある。

まず1タイプだけ、無料で見せる。

## 火タイプ（丙・丁）のLINE攻略

火タイプは感情が豊かで、ノリを大事にする人。テンションの波が大きくて、盛り上がっている時と落ちている時の差が激しい。

**このタイプにLINEを送る時の鉄則:**

**1. リアクションを大きくする**
「そうなんだ」→ ×
「えっ、マジで！？すごいね！」→ ○

火タイプは自分の話にリアクションが薄いと、一気に興味を失う。「この人、つまんないな」と思われたら終わり。少し大げさなくらいでちょうどいい。

**2. 短文・テンポ重視**
長文を送ると読む気が失せる。LINEはチャットのテンポで。ポンポン送り合う感覚が大事。

**3. 絵文字・スタンプを活用**
火タイプは感情表現が好き。文字だけの無機質なLINEは「冷たい」と感じる。

**4. 盛り上がっている時に次の約束をする**
テンションが高い時に「じゃあ今度○○行こうよ！」が最強。冷めてから誘っても「うーん、また今度」になる。

**NGワード:**
- 「まあいいんじゃない」（←興味なさそうに聞こえる）
- 「別に」（←冷たすぎる。火が消える）
- 長文の説教（←テンション下がって二度と返事こない）

**既読スルーされた時の復活LINE:**
→ 「そういえばさ、」で始まる新しい話題をぶっ込む。
火タイプは新しい刺激に反応する。重い追いLINEは絶対ダメ。「返事まだ？」は死亡フラグ。何事もなかったかのように、面白い話題で仕切り直す。

## 木タイプ（甲・乙）のLINE完全攻略

木タイプは自立心が強く、自分のペースを何より大事にする人。束縛を嫌い、自分の時間と空間を確保したいタイプ。

### 返信率が上がるメッセージ例

**1. 「報告」ではなく「共有」**
- ×「今日何してた？」（←監視されてる感）
- ○「今日こんな面白いことがあってさ」（←自分の話を共有する形。返信のプレッシャーがない）

**2. 相手の専門分野や得意なことを「聞く」**
- ○「○○のことで聞きたいんだけど」（←頼られるのは嬉しい。ただし頻度は控えめに）
- ○「○○についてどう思う？」（←意見を求められるのが好き）

**3. 自立していることを見せる**
- ○「今日○○やってみた！」（←自分で行動できる人だと伝わる）
- ○「一人で○○行ってきたけど、よかったよ」（←依存してない感が出る）

**4. 長文OK（ただし中身がある場合）**
- 木タイプは中身のある話が好き。薄い会話より、ちゃんとした話題の方が返事が来る。
- ただし「感情の垂れ流し」はNG。「考え」や「意見」を書く。

**5. 間を空ける勇気**
- ○ 毎日送らない。2〜3日空いても焦らない。
- 木タイプは自分のペースを乱されると距離を取る。「連絡がないこと」を不安に思わないのが大事。

### NGワード・NG行動
- 「なんで返事くれないの？」（←最大の地雷。追い詰められると逃げる）
- 「寂しい」を連発（←重い。依存に見える）
- 毎日の「おはよう」LINE（←義務感が出る。自分のリズムを崩される）
- 電話を突然かける（←事前に「電話していい？」と聞くのがマナー）

### ベストな送信タイミング
- **平日の夜（21時〜23時）:** 仕事が終わって自分の時間に入った後。リラックスしている。
- **土曜の昼:** 自分の時間を楽しんでいる最中。テンション高め。
- **避けるべき:** 朝の通勤中、仕事中、日曜の夜（翌日の仕事モードに入っている）

### 既読スルーされた時の復活LINE
→ **3日は放置。** 焦って追いLINEを送ると一気に冷める。
→ 3日後に「全然関係ない話題」を1通だけ送る。
→ 例：「そういえば、前言ってた○○ってどうなった？」（相手の話を覚えていたことが伝わる）
→ これで返事が来なければ、さらに1週間空ける。木タイプの既読スルーは「嫌い」じゃなく「今じゃない」のことが多い。

### 脈ありを確認するテストメッセージ
→ 「今度○○あるんだけど、一緒にどう？」と具体的な提案を送る。
→ 脈あり：日程の相談が始まる or 「行きたい！」と即答
→ 脈なし：「ちょっと予定見てみる」→ 返事なし
→ 木タイプは興味があれば行動が早い。曖昧な返事が続くなら、今はタイミングじゃない。

## 土タイプ（戊・己）のLINE完全攻略

土タイプは安定志向で、誠実さを大事にする人。信頼できる人には心を開くけど、軽いノリの人には警戒する。

### 返信率が上がるメッセージ例

**1. 誠実さが伝わる文章**
- ○「この前のこと、ちゃんと考えてたんだけど」（←真剣に向き合ってる感）
- ○「ありがとう。あの時助かった」（←感謝を言葉にする）

**2. 日常の小さな報告**
- ○「今日のお昼、○○食べたよ。美味しかった」（←日常を共有する安心感）
- ○「○○が咲いてた。春だね」（←穏やかな話題が好き）
- ※火タイプにはNGな「今日どうだった？」が、土タイプには○。安心感になる。

**3. 「あなたのおかげ」を入れる**
- ○「あなたに言われた通りにしたら、うまくいったよ」（←貢献を実感できる）
- ○「○○さんがいてくれると安心する」（←存在を肯定する）

**4. 写真を送る**
- ○ 食べ物、風景、日常の一コマ。
- 土タイプは「生活」を大事にするから、日常の写真に親近感を持つ。

**5. 返信を急かさない**
- 土タイプはマイペース。返事が遅くても、ちゃんと考えてくれている証拠。
- 「了解！」「いつでもいいよ」のスタンスが信頼につながる。

### NGワード・NG行動
- 「ノリ悪いなー」（←ペースを否定される）
- チャラい言動（←信頼を失う最速ルート）
- 約束の変更を繰り返す（←一発で信頼が崩壊する）
- 他の異性の話を匂わせる（←嫉妬ではなく「信頼できない人」認定される）

### ベストな送信タイミング
- **夜の21時頃:** 一日が終わって落ち着いた時間。
- **休日の午前中:** ゆっくりしている時間帯。
- **避けるべき:** 深夜（不信感）、忙しそうな時間帯

### 既読スルーされた時の復活LINE
→ **焦らない。2日待つ。**
→ 2日後に「重くない日常LINE」を送る。
→ 例：「今日めっちゃ寒くない？体調気をつけてね」（さりげない気遣い）
→ 土タイプの既読スルーは「忙しい」か「返す内容を考えている」のどちらか。追い詰めなければ自然と返ってくることが多い。

### 脈ありを確認するテストメッセージ
→ 「今度ご飯でもどう？○○が美味しいらしいんだけど」
→ 脈あり：「いいね！いつにする？」とすぐ日程の話になる
→ 脈なし：「いいね」だけで終わる or スルー
→ 土タイプは社交辞令が下手。「いいね」と言ったら本気で「いいね」と思っている。逆に、曖昧にされたら脈は薄い。

## 金タイプ（庚・辛）のLINE完全攻略

金タイプはこだわりが強く、筋が通っていることを大事にする人。プライドが高く、自分の美学がある。

### 返信率が上がるメッセージ例

**1. ストレートに伝える**
- ○「会いたいんだけど、来週空いてる？」（←回りくどいのが嫌い）
- ○「正直に言うと、○○だと思う」（←率直さを好む）

**2. センスを褒める**
- ○「その選び方、センスいいね」（←こだわりを認められるのが嬉しい）
- ○「○○さんのおすすめなら間違いない」（←審美眼を信頼される）

**3. 自分の意見をちゃんと持つ**
- ○「自分は○○だと思うんだけど、どう思う？」（←対等な議論ができる人に惹かれる）
- ×「何でもいいよ〜」「任せる！」（←優柔不断に見える。興味を失う）

**4. 特別感を出す**
- ○「○○さんにだけ聞きたいことがあって」（←選ばれた感）
- ○「これ、○○さんなら分かると思って」（←理解者として認められたい）

**5. 文章の質を意識する**
- 金タイプは文章の雑さに敏感。誤字脱字、雑な言葉遣いで印象が下がる。
- 丁寧すぎる必要はないけど、「ちゃんとした人」感は出す。

### NGワード・NG行動
- 「まあいいじゃん、細かいこと気にしないで」（←こだわりを軽視される）
- スタンプだけの返信を連発（←雑に扱われてると感じる）
- 他の人に言ったことと同じことを言う（←「それ誰にでも言ってるでしょ」と見抜く）
- 嘘をつく（←一度嘘がバレたら二度と信頼されない）

### ベストな送信タイミング
- **夜の20時〜22時:** 自分の時間に入ってリラックスしている。
- **平日ランチ（12時〜13時）:** 意外とこの時間に返す人が多い。
- **避けるべき:** 朝イチ（テンションが合わない）、金曜夜（自分の予定を優先したい）

### 既読スルーされた時の復活LINE
→ **1日待って、質問形式で送る。**
→ 例：「○○さん的に、AとBどっちがいいと思う？」（意見を求められると答えたくなるタイプ）
→ 「返事まだ？」系は絶対NG。プライドを傷つける。
→ 金タイプの既読スルーは「今返す気分じゃない」か「返すに値する内容じゃない」。内容の質を上げると返ってくる。

### 脈ありを確認するテストメッセージ
→ 「○○さんが好きそうな店見つけたんだけど、今度行かない？」
→ 脈あり：「どこ？」と食いつく or 具体的な日程を提案してくる
→ 脈なし：「へぇ」で終わる
→ 金タイプは興味があれば自分から動く。受け身のままなら、今は脈なし。

## 水タイプ（壬・癸）のLINE完全攻略

水タイプは自由を愛し、束縛を嫌う人。好奇心旺盛で、色んなことに興味がある。掴みどころがないように見えて、実は感受性が高い。

### 返信率が上がるメッセージ例

**1. 意外性のある話題**
- ○「今日こんな変なことがあってさ（笑）」（←面白い話に食いつく）
- ○「○○って知ってる？」（←知的好奇心を刺激する）
- ×「お疲れ様」「今日もありがとう」（←定型文にはほぼ反応しない）

**2. 自由を尊重するスタンス**
- ○「暇な時でいいんだけど」（←プレッシャーをかけない）
- ○「返事いつでもいいよ〜」（←縛らない。これだけで好感度が上がる）

**3. ミステリアスさを出す**
- ○「面白いこと思いついたんだけど、会った時に話すね」（←気になる。会いたくなる）
- ○ あえて全部を伝えない。想像の余白を残す。

**4. 写真や動画を活用**
- ○ 面白い風景、珍しい食べ物、変なもの。
- 水タイプは「新しいもの」に反応する。文字より画像の方がテンポよく返事が来る。

**5. 追わない**
- これが最大のポイント。水タイプは追われると逃げる。
- 自分も楽しそうに過ごしている感を出す。「あなたがいなくても楽しいよ、でも一緒だともっと楽しい」がベストな距離感。

### NGワード・NG行動
- 「なんで連絡くれないの？」（←束縛。一発退場レベル）
- 毎日同じ時間にLINE（←ルーティン化すると逃げたくなる）
- 長文の感情LINE（←重い。読む気が失せる）
- 予定を細かく聞く（←監視されてる気分になる）

### ベストな送信タイミング
- **不定期が一番。** 決まった時間に送らない方がいい。
- **深夜（23時〜1時）:** 意外と夜型が多い。この時間帯にポンと送ると返ってくることがある。
- **避けるべき:** 毎日同じ時間（パターン化を嫌う）

### 既読スルーされた時の復活LINE
→ **追わない。** これが全て。
→ 1週間くらい何も送らない。
→ その後、「全然関係ない面白い話題」をポンと送る。
→ 例：ニュースのリンク、面白い動画、「これ見て（笑）」
→ 水タイプは「楽しい人」のLINEには返す。「重い人」のLINEは無視する。楽しさを取り戻すのが鍵。

### 脈ありを確認するテストメッセージ
→ 「○○やってみたいんだけど、一緒にやらない？」（体験型の誘い）
→ 脈あり：「面白そう！いつ？」と即反応
→ 脈なし：「いいね〜」→ 以降スルー
→ 水タイプは興味があれば反応が早い。返事が遅い＝優先順位が低い、とシンプルに判断していい。

## まとめ：LINEは"何を送るか"じゃなくて"誰に送るか"で変えろ

ネットに転がっている「モテるLINEの送り方」は、全員に同じことを言っている。

「追いLINEはダメ」
「即レスしすぎない」
「ミステリアスに」

これ、半分正解で半分ウソ。

タイプによっては即レスの方が好感度上がるし、ミステリアスにしたら「冷たい」と思われることもある。
